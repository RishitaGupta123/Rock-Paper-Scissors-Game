# 🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using **HTML**, **CSS**, and **JavaScript**. Play against the computer and enjoy a clean, responsive user interface with real-time score updates.

## 🚀 Live Demo

👉 https://rishitagupta123.github.io/Rock-Paper-Scissors-Game/

## 📌 Features

- 🪨 Rock, 📄 Paper, ✂️ Scissors gameplay
- 🤖 Random computer choice
- 📊 Live score tracking
- 🎨 Clean and responsive UI
- ⚡ Fast and lightweight
- 📱 Works on desktop and mobile browsers
- 🔄 Play unlimited rounds

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 How to Play

1. Click on Rock, Paper, or Scissors.
2. The computer will randomly choose its move.
3. The winner is decided automatically.
4. Scores are updated after every round.
5. Keep playing to beat the computer!

## 🎮 Game Rules

- 🪨 Rock beats Scissors
- ✂️ Scissors beats Paper
- 📄 Paper beats Rock
- Same choice = Draw
